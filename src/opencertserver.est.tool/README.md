# opencertserver.est.tool

This project provides a command-line tool for interacting with EST (Enrollment over Secure Transport) servers. It allows users to enroll, re-enroll, and manage certificates from the terminal, supporting automation and scripting scenarios.

## Functionality
- Command-line interface for EST operations
- Supports certificate enrollment and re-enrollment
- Designed for automation and scripting

## Dependencies
- CommandLineParser (for argument parsing)
- opencertserver.est.client (EST client logic)

Use this tool to automate EST certificate management tasks.
