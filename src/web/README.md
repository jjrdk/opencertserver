# web

This project contains the Angular-based web frontend for OpenCertServer. It provides a user interface for certificate enrollment, management, and administration, integrating with the backend services via REST APIs.

## Functionality
- User interface for certificate authority operations
- Supports certificate enrollment, revocation, and inventory
- Integrates with ACME, CA, and EST backend services

## Dependencies
- Angular framework
- Angular Material (UI components)
- Various npm packages for HTTP, forms, and UI

Use this project to provide a web-based management interface for OpenCertServer.
