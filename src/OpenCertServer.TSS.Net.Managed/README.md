# OpenCertServer.TSS.Net.Managed

This is a copy of the TSS.Net source code, which is a .NET wrapper around the TSS library. The TSS library is a C 
library that provides a high-level API for working with TPM 2.0 devices. The TSS.Net wrapper allows you to use the TSS 
library from .NET applications.

It pulls in some classes from BouncyCastle.
